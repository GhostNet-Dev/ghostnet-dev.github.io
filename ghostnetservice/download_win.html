Ghost Studio

Ghost Studio는 MasterNode를 구성하는데 필요한 요소들을 최신상태로 유지해주며 관리를 할 수 있는 GUI를 제공합니다. Local에서 Generative AI나 NFT 생성등을 제한없이 사용할 수 있습니다.

Install

NodeJS + Electron 기반으로 개발되었습니다. Test는 Window에서 진행했고 아마도 Mac이나 Linux Desktop도 될거라 생각됩니다만...

GUI Interface는 Server와 대부분 동일하나 개발자 능력 부족으로 확장 기능에서 차이가 있을 수 있습니다. (예: Desktop에서만 Terminal 제공)
