ghost.chat.net@gmail.com

Install

먼저 docker가 설치되어야합니다. 전달받은 주소로 git을 clone합니다.

git pull origin master

그리고 docker build를 시작합니다.

docker volume create --name ghost
docker build . -t ghost

Run

build가 정상적으로 완료되면 docker를 다음 명령어로 실행합니다.

docker run --name ghostd -v ghost:/usr/src/app --net=host -p 8090:8090 -p 8091:8091 -p 50128:50128/udp -p 58080:58080 ghost

localhost:8090에 웹브라우저로 접속하면 관리자 화면을 확인할 수 있습니다.
